# Guessing Game
Welcome to the Guessing Game! This interactive web application challenges players to guess a number within a specified range. Built using HTML for structure, CSS for styling, and JavaScript for dynamic behavior, the game prompts users to input their guesses, providing feedback on whether the guess is too high, too low, or correct. To play, simply open the index.html file in your web browser. The game's code is well-commented, allowing for easy customization and extension.
